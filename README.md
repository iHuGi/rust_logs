# RustyResults 🦀

A simple playground project to practice **Error Handling** and **Pattern Matching** in Rust. This is part of my journey through the Stephen Grider Rust course on Udemy.

## 🚧 Work in Progress
This repo is a "construction zone" where I'm breaking things and fixing them to understand how Rust's safety features actually work.

### What's inside?
- **Result Handling**: Practicing the `Result<T, E>` type to handle success and failure gracefully.
- **Pattern Matching**: Using `match` blocks to handle different data outcomes (no more messy `if/else` logic).
- **Validation Logic**: Simple tools like email validation and safe division (no more dividing by zero!).
- **WSL Dev**: Built and tested entirely within **Ubuntu on WSL2**.

## How to run
Since this is a standard Cargo project, just use the terminal:

```bash
# Clone it


# Run it
cargo run -q
```
